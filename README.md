# React Burger

React Burger is a web application for ordering burgers online. It was built using React, Redux, and React DnD.

See on [github pages](https://boolishta.github.io/react-burger/)

## Installation

To install React Burger, clone the repository and run `npm install`.

## Structure

- `src`: Contains the source code for the React Burger app.
- `public`: Contains static files such as images and HTML templates.
- `package.json`: Includes the dependencies and scripts for the project.
